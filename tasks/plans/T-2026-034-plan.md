# Plan: T-2026-034 — Create mastery star display component

## Task

A reusable component that displays 0-5 mastery stars for a topic, with color-coded glow matching the mastery glow colors from the style guide. Inputs: `stars` (0-5), `size` ('sm' | 'md' | 'lg'). Accessible with aria-label. Unit tested.

## Architecture

**Component Design:** A standalone Angular component using signal inputs (`input()`) and `computed()` signals, following the exact patterns established by `XpProgressBarComponent` and `StationCardComponent`. The component renders 5 `<lucide-icon name="star">` elements in a row, applying filled/empty styling and a glow color CSS variable based on the `stars` input.

**Key Decisions:**

1. **Lucide `star` icon for both filled and empty states.** The `Star` icon is already registered in `APP_ICONS`. Filled stars use `fill: currentColor` + `stroke: currentColor`; empty stars use `fill: transparent` + `stroke: currentColor` with reduced opacity. This avoids needing a separate "star-outline" icon import.

2. **CSS custom property for glow color.** The component sets `--mastery-glow` on the host element via a computed signal, mapping stars count to the existing `--nx-mastery-N` CSS variables from the design token system. The SCSS uses this variable for `color`, `filter: drop-shadow()`, and the pulse animation.

3. **Pulse animation for 5 stars only.** A CSS `@keyframes` pulse animation is applied via a host class `mastery-stars--pulse` only when `stars === 5`. Respects `prefers-reduced-motion`.

4. **Size variants via host classes.** Size maps to icon pixel size: sm=16, md=20, lg=24. Applied as a host class (`mastery-stars--sm`, etc.) and a `--mastery-star-size` CSS variable that is forwarded to Lucide's `[size]` input.

5. **No @for with index; use computed array.** A computed signal produces an array of 5 booleans indicating filled/empty, which the template iterates with `@for`. This keeps the template declarative and testable.

6. **Test host pattern.** Tests use the established `TestHost` wrapper + `createComponent()` pattern with `ICON_PROVIDERS` for Lucide icon rendering, matching `icons.spec.ts`.

## Detailed File Specifications

### mastery-stars.ts (~50 LOC)

```
Imports: Component, computed, input from @angular/core
         LucideAngularModule from shared/icons

Selector: nx-mastery-stars
Standalone: true (default in Angular 21)
Imports: [LucideAngularModule]
styleUrl: ./mastery-stars.scss

Host bindings (via `host` metadata):
  'role': 'img'
  '[attr.aria-label]': 'ariaLabel()'
  '[class.mastery-stars--sm]': 'size() === "sm"'
  '[class.mastery-stars--md]': 'size() === "md"'
  '[class.mastery-stars--lg]': 'size() === "lg"'
  '[class.mastery-stars--pulse]': 'stars() === 5'
  '[style.--mastery-glow]': 'glowColor()'

Inputs:
  readonly stars = input<number>(0)       // 0-5
  readonly size = input<'sm' | 'md' | 'lg'>('md')

Computed:
  iconSize = computed(() => ({ sm: 16, md: 20, lg: 24 })[this.size()])

  starStates = computed(() => {
    const count = Math.min(5, Math.max(0, Math.round(this.stars())));
    return Array.from({ length: 5 }, (_, i) => i < count);
  })

  glowColor = computed(() => {
    const level = Math.min(5, Math.max(0, Math.round(this.stars())));
    // Don't set glow at 0 stars — let the CSS fallback (--nx-color-corridor) activate
    return level === 0 ? null : `var(--nx-mastery-${level})`;
  })

  ariaLabel = computed(() => {
    const count = Math.min(5, Math.max(0, Math.round(this.stars())));
    return `${count} out of 5 stars mastery`;
  })

Template:
  @for (filled of starStates(); track $index) {
    <lucide-icon
      name="star"
      [size]="iconSize()"
      [class.mastery-stars__star--filled]="filled"
      [class.mastery-stars__star--empty]="!filled"
      aria-hidden="true" />
  }
```

### mastery-stars.scss (~55 LOC)

```
:host {
  display: inline-flex;
  align-items: center;
  gap: var(--nx-space-1);   // 4px between stars
  color: var(--mastery-glow, var(--nx-color-corridor));
}

// Filled stars: solid fill + stroke, with glow color
// No ::ng-deep needed — Lucide renders SVG in the light DOM (confirmed by icons.spec.ts)
.mastery-stars__star--filled {
  svg {
    fill: currentColor;
    stroke: currentColor;
  }
  filter: drop-shadow(0 0 4px currentColor);
}

// Empty stars: outline only, muted
.mastery-stars__star--empty {
  opacity: 0.25;
  svg {
    fill: transparent;
    stroke: currentColor;
  }
}

// Size variants (gap adjustments)
:host(.mastery-stars--sm) {
  gap: 2px;
}

:host(.mastery-stars--lg) {
  gap: var(--nx-space-2);   // 8px
}

// Pulse animation for 5-star mastery
// Animate color opacity instead of host filter to avoid stacking context conflict with child drop-shadows
:host(.mastery-stars--pulse) {
  animation: mastery-pulse 2s ease-in-out infinite;
}

@keyframes mastery-pulse {
  0%, 100% { opacity: 0.85; }
  50% { opacity: 1; }
}

@media (prefers-reduced-motion: reduce) {
  :host(.mastery-stars--pulse) {
    animation: none;
  }
}
```
